| <a name="input_name"></a> [name](#input\_name) | The name of the Virtual Network. | `string` | n/a | yes |
| <a name="input_resource_group_name"></a> [resource\_group\_name](#input\_resource\_group\_name) | The name of the Resource Group in which to create the Virtual Network and DDoS Plan (if enabled). | `string` | n/a | yes |
